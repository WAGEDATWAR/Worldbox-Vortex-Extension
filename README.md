# WORLDBOX MOD LOADER V1.0.0
*Created by WAGEDATWAR*

# 1) INTRODUCTION

Worldbox has a horrible history with authors adopting mod managers, meaning a large majority of mods are hosted on gamebanana and the user is expected to manage them from there. My goal with this extension is primarily to allow gamebanana mods to be managed and checked for updates in a centralized place. Other features may come later, but this is the core aim of this extension. I have almost no programming experience and know even less about JavaScript, most of this was copy and paste.

# 2) REQUIREMENTS

* A legal copy of Worldbox(must be Steam)
* NeoModLoader: Worldbox's primary mod loader, required for content and code mods. You can download it from their [Github](https://github.com/WorldBoxOpenMods/ModLoader/releases).
